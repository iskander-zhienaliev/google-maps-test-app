# TestApp_google-maps-react


### Чего и как делать для начала работы?

##### 1. Скачиваем репу  
>`git clone <путь до репозитория с кодом> <куда ставим>`

##### 2. Ставим зависимости
>`npm install`

##### 3. Запускаем проект
>`npm run start`


##### 4. Открываем в браузере
> `http://localhost:8080`
